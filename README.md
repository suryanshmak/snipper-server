# snipper-server

What a efficient go web server looks like. Serves [snipper-client]('https://github.com/suryanshmak/snipper-client')

1. Prerequisites: **Go**

2. Setting up a local environment

```
git clone https://github.com/suryanshmak/snipper-server
```

```
cd snipper-server
```

```
go install
```
